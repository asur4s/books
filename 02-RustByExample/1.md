## 注释

注释和 C 系一样，还有就是每个语句后有分号。最大的改变就是关键词不太一样。
```rust
fn main() {
    // This is an example of a line comment
    let x = 5 + /* 90 + */ 5;
    println!("Is `x` 10 or 100? x = {}", x);
}
```
没有返回值，没有参数列表，也没有包含头文件。
